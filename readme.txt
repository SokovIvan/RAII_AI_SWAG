AI SWAG

Проект для анализа звуковых файлов по заявкам ЖКХ и составление краткого описания и классификации.

Используются нейронные сети для анализа и Django для графического интерфейса.

Инструкция по сборке:

1) Скачать и распаковать архив
2) Установить Python 3.10 и необходимые библиотеки из requirements.txt
3) Установить ffmpeg
4) Запустить RAII_AI_SWAG\UI\backend\manage.py runserver (python manage.py runserver)
5) Открыть браузер по http://127.0.0.1:8000/
